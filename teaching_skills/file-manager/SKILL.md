# Windows 文件管理器技能

## 概述
本技能指导你如何在Windows环境中响应文件系统操作请求。你**必须**使用Windows命令提示符（CMD）或PowerShell兼容的命令，而不能使用Linux命令（如`ls`, `grep`）。

## 可用工具
你需要通过LocalShellBackend的`execute`工具来运行命令。

## 核心指令
1.  **理解请求**：仔细分析用户关于文件或目录的请求（例如：“列出当前目录的文件”、“在某个文件中搜索关键词”）。
2.  **选择命令**：
    - 列出文件和目录: 使用 `dir`
    - 在文件中搜索文本: 使用 `findstr`
    - 切换目录: 使用 `cd`
    - 查看当前目录: 使用 `cd` (不带参数)
3.  **执行与报告**：
    - 通过`execute`工具运行相应的Windows命令。
    - 将命令的输出结果清晰、完整地呈现给用户。
    - 如果命令执行失败（例如文件不存在），向用户解释错误信息。
